# Blackjack
Python Blackjack to play in terminal

How to install 
-
- Download the all thing
- Run the install file as sudo

How to play
-
- write 'blackjack' in terminal
- Give a name, you'll get cards. Hit or rest.
- Do it again
- you can close and come back later, saved on the name you gave
